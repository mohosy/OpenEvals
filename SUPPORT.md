# Support

## Where to ask questions

- Usage questions and setup help: GitHub Discussions
- Bugs and reproducible defects: GitHub Issues
- Security concerns: follow [SECURITY.md](./SECURITY.md)

## Good support requests

Include:

- The suite YAML or a reduced reproduction
- The model names used
- Whether the run happened in the UI, CLI, or GitHub Actions
- Relevant logs or screenshots

## Benchmarks and examples

If you want OpenEvals to cover your domain, open a benchmark-suite issue or start a discussion with a public example.
